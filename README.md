# umamusume-loop-factor
因子ループおぼえられましぇん

```shell
$ umamusume-loop-factor % go run main.go トウカイテイオー スペシャルウィーク エルコンドルパサー テイエムオペラオー
----- 子->親の組み合わせパターン構築開始 -----
トウカイテイオー
   ├─  スペシャルウィーク
   └─  エルコンドルパサー
テイエムオペラオー
   ├─  トウカイテイオー
   │         ├─スペシャルウィーク
   │         └─エルコンドルパサー
   └─  スペシャルウィーク
エルコンドルパサー
   ├─  トウカイテイオー
   │         ├─スペシャルウィーク
   │         └─エルコンドルパサー
   └─  テイエムオペラオー
             ├─トウカイテイオー
             └─スペシャルウィーク
スペシャルウィーク
   ├─  エルコンドルパサー
   │         ├─トウカイテイオー
   │         └─テイエムオペラオー
   └─  テイエムオペラオー
             ├─トウカイテイオー
             └─スペシャルウィーク
トウカイテイオー
   ├─  スペシャルウィーク
   │         ├─エルコンドルパサー
   │         └─テイエムオペラオー
   └─  エルコンドルパサー
             ├─トウカイテイオー
             └─テイエムオペラオー
----- 組み合わせパターン構築完了 -----
```
